# 如何参与分享爆料革命 #  
## 1. 提醒 ##  
1. I2P下载是需要有耐心等的，没有正常互联网那么快，现阶段参与人数较少，下载速度不太可能快了。  
2. 墙外战友最好能打开 "I2P匿名网络"的同时，打开"公共IP网络(不匿名)"，这样能够向所有 Bittorrent 软件分享，保持所有内容在正常互联网的传播。  
3. 本教程会根据战友的反馈不断修改，有兴趣的战友持续关注。  
4. 不要一次只下载一个，最好多下载几个(网上推荐 20 个)，这样能够保证速度。东边不亮，西边亮。也能给大家分享。
## 2. 安装软件 ##  
本软件支持 WinXP / Win7 / Win8 / Win10 / macOS / linux / unix / Android / Android TV，目前还没发现 IOS 版本。几乎覆盖了战友电子设备，就算有个智能电视都能做安全分享。

### 2.3. 软件设置 ###  
macOS安装后可能是英文界面，需要设置为中文，菜单中找到 Tools -> Options 打开 Preferences 窗口，右侧选择 Interface -> Language，向下滚动，选择 "中文(中国) - Chinese(China)"。  
一但转换成中文界面，后边按照以下流程，与 Win 基本一致。
#### 2.3.1. 开启高级模式 ####  
因为这个软件为了防止新手设置错误，新安装后的模式为 "新手"，好多功能无法使用，要改模式为 "高级"。  
菜单中找到 工具 -> 选项 打开 选项窗口，选择 模式 -> 用户熟练度 -> 高级。然后点 "保存"按钮。  
#### 2.3.2. 开启 I2P Helper ####  
某些版本默认不开启此模块，所以要确认一下。  
菜单中找到 工具 -> 选项 打开 选项窗口，选择 插件 -> I2P Helper，确认 "Enable" 为选择状态。  
菜单中找到 工具 -> 选项 打开 选项窗口，选择 连接  网络  选择默认许可的点对点数据通信网络，选择 "公共IP网络" 和 "I2P匿名网络"。此项选择可以自动以 I2P匿名网络开始下载。  
保存后退出 选项窗口。  
### 2.4. 等待 I2P 连网 ###  
因为 I2P 是隐藏网络，没有中央服务器，所以安全的同时也造成启动时连接需要点时间，这个时间大概是 5 到 10 分钟。  
菜单中找到 查看 -> I2P Helper，在窗口侧边栏 -> 插件和附加内容 下打开 I2P Helper 插件。  
<img src="/img/007.gif" width="80%" height="80%">  
窗口右侧 "Mix DHT Stats" 标签中的内容初始值比较小，如：用户、叶子、节点数的值都为1，等待 5 到 10 分钟会快速增加，达到几百或几千，甚至过万，这说明你已经连接到 I2P网络。  
### 2.5. 订阅 RSS ###  
这个订阅，我用个人电脑24小时开机做为订阅服务器，为战友分享订阅。  
订阅地址：http://4zlwfvcvu4hu3k7xigtwczo4betkhzdkpip54c24ghotiamtv6ra.b32.i2p/rss-miles.xml  
<img src="/img/008.gif" width="80%" height="80%">  
窗口侧边栏 -> 内容搜索 -> 订阅，点击订阅后的加号按钮。弹出订阅窗口。  
<img src="/img/009.gif" width="80%" height="80%">  
点击窗口中"创建新的订阅"按钮。  
<img src="/img/010.gif" width="80%" height="80%">  
在此界面中的 "RSS 订阅" 标签内的 "在下面输入或粘贴 URL" 下的输入框内填写上边的订阅地址。点击保存。  
等待大概2秒关闭此窗口。在主窗口的"订阅"中出现上边增加的链接。在链接上点击鼠标右键  现在更新。等待一段时间会出现一个下载列表。  
<img src="/img/011.gif" width="80%" height="80%">  
在此列表中点击每一行中的 "下载"，弹出"操作进行中窗口"。此窗口可以关闭，但下载已经开始。最初是下载种子，种子下载后自动下载文件。  
当所有 "下载" 都点击后，在侧边栏中"我的媒体库"或"正在下载"中全选所有下载。点击右键，弹出右键菜单。  
<img src="/img/011-2.gif" width="80%" height="80%">  
这个订阅会根据服务器的订阅变化自动更新，更新的内容"未打开"列会列是深蓝色，一但出现这样的说明有更新，你只要下载就可以了。  
<img src="/img/012.gif" width="80%" height="80%">  
右键菜单 -> 控制 -> 强制开始(第一行第二个按钮)。确保下载数量太多时，所有下载都在下载状态中。  
如果下载多个，可全选后再点右键(Win:Ctrl+A / Mac:Command+A)。  
<img src="/img/012-1.gif" width="80%" height="80%">  
右键菜单 -> 控制 -> 网络 -> I2P匿名网络。确保所有下载都通过I2P网络进行下载。因为，我做的分享都是通过I2P网络进行的。  
如果下载多个，可全选后再点右键(Win:Ctrl+A / Mac:Command+A)。  
这种下载跟 BT 下载一样，不会马上开始，需要渐进性找节点，开始时需要一段时间，当找到节点时，速度慢慢加快。当节点多的时侯，速度提升。

## 4. 结尾语 ##  
这个项目只分享了VOG两个月的音频，如果参与此项目，最好能够保持24小时开机。I2P占用的网络带宽不是很大，这个软件只占用50K的上行和下行带宽。如果战友有个旧电脑就能参与分享。你在下载过程中也是向墙内战友分享。下载后最好不是马上删除看过的文件。保持一个分享时间。我的建议是一周。
